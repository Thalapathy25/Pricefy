About

Pricefy is a web app which has a Machine Learning model running at the backend. The purpose of developing this app is to predict price of a used car depending of it's model year, present selling price, Kilometers driven, Fuel type, No of owners, Seller type and Transmission type. This model is based on the car data in Used Car price dataset. This Project is created using python's scikit-learn, fastapi, numpy and joblib packages. You can also try Pricefy app using interactive documentation such as Swagger UI and Redoc UI. You can also download OpenAPI specification as in JSON format. The codes for this project can be checked in my github repository.
